> ## pod 概念

#

### 1.pod 基础概念

```bash
    1.最小部署的单元
    2.pod是由一个或多个容器组成.
    3.pod你可以抽象成一个进程组的概念
    4.一个pod中容器共享网络命令空间.
    5.pod是短暂的

```

### 2.pod 存在意义

```bash
    1.创建容器使用docker,一个docker对应是一个容器,一个容器运行一个进程.

```
