##  四、Redis入门

```shell

1. redis是什么？
    -Redis（Remote Dictionary Server）即远程字典服务
    是一个开源的使用ANSI C语言编写、支持网络、可基于内存亦可持久化的日志型、Key-Value数据库，并提供多种语言的API。

2. redis 能干嘛？
    -内存存储,持久化,内存中是断电即失,所以说持久化很重要(rdb,aof)
    -效率高,可以用于高速缓存.
    -发布订阅系统
    -地图信息分析
    -计时器,计数器(浏览量)
    
3. 特性
    -多样性数据类型
    -持久化
    -集群
    -事务


 4. 常用网站
      -官网:   https://redis.io/
     -中文网:   https://www.redis.cn/
     -github:  https://github.com/dmajkic/redis

```
