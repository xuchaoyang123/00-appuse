>## 十五、redis发布订阅
#

### 1.基本概念
```bash
  概念:
    #redis发布订阅(pub/sub)是一种消息通信模式,发送者(pub)发送消息,订阅者(sub)接收消息.
    #redis客户端可以订阅任意数量的频道.
    #第一个概念: 生产者  第二个概念:频道  第三个概念:订阅者
    #生产者=老师,频道=课程,订阅者=同学.
  应用场景:
  1. 实时消息系统(B站的系统消息,其实就是默认全站会员订阅了官网的频道)
  2. 事实聊天(频道当做聊天室,将信息回显所有人即可.)
  3. 订阅,关注系统都是可以的.
    

```



### 2.操作命令
```bash
    #窗口1:订阅者订(同学)
    127.0.0.1:6379> ubscribe pindao01                   #订阅者订阅频道,然后会一直监听着.

    #窗口2:生产者(老师)
    127.0.0.1:6379> publish pindao01 "hello word"       #生产者发布消息和内容.  

    #窗口1:订阅者(同学)
    127.0.0.1:6379>                                     #监听窗口可以看到是哪个频道有什么消息.
    
    #窗口1:订阅者(同学)
    127.0.0.1:6379> unsubscribe pindao01                #退订频道.


```
