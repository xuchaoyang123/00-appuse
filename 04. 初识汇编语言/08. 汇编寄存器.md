### 3. 通用寄存器

```shell 
#寄存器
  -存储数据: cpu >内存> 硬盘
  -32位CPU: 8  16 32 
  -64位CPU: 8 16  32  64 

#通用寄存器
   -32位的通用寄存器只有8个
   -存数值最大范围: 0 ~FFFFFFFF

    EAX  
    ECX
    EDX
    EBX
    ESP
    EBP
    ESI
    EDI 

#计算机如何像寄存器存值
 
 mov指令:

    #可以将数值写到寄存器
    #可以将寄存器的数值写到寄存器

    mov 存的地址,存的数.
    mov 存的地址1,存的地址1

    #32位修改
    mov ecx,1      #修改ecx寄存器数值为1
    mov edx,ecx    # 将ecx的数值复制到edx里.

    #8位修改
    mov al,1   


```

### 4. 不同的寄存器

```shell
    #L代表底8位
    #H代表高8位
    #高八位+低八位=16位  0001 + 0010 = 0101 ？？？
    #除了这些寄存器之外,其他寄存器每一位都有自己特定的功能.

      FFFF   FF 0000 0000
32位   16位  8位

EAX    AX    AL
ECX    CX    CL
EDX    DX    DL 
EBX    BX    BL <-----L代表底8位
ESP    SP    AH <-----H代表高8位
EBP    BP    CH
ESI    SI    DH
EDI    DI    BH



例:
  
1. 正常16位置,写入数据.   AX 16位命令

    EAX:00000000  #32位原值  
    mov ax,-1     #写入-1 ,ax是16位,所以只会涉及到后面四个0的数据.
    EAX:0000ffff  

2.  低八位,写入数据     AL 低八位命令
    ECX:00000000 # #32位原值 
    mov al,1     # 存入1.
    CX:00000001  #结果  <---------低八位 0001


3.  高八位,写入数据     AH 低八位命令
    ECX:00000001   # #32位原值 
    mov ah,1       # 存入1.
    CX:00000101    #结果 <---------高八位 0101  


```
